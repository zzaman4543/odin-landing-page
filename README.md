# Landing Page Project

[Landing Page](https://www.theodinproject.com/lessons/foundations-landing-page) is the second project in the [The Odin Project](https://www.theodinproject.com)'s [Foundations](https://www.theodinproject.com/paths/foundations/courses/foundations) path.

## Skills Gained

Completing this project taught me how to develop layouts solely using flexbox. I've learnt about core parts of flexbox, such as flex items, flex containers,
growing, shrinking, axes, and alignment.
